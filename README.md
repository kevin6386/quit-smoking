1. App 简介
戒烟助手是一款以“本地记录 + 日历打卡 + 省钱统计 + 戒烟科普”为核心的轻量应用。

它的重点不是喊口号，而是把戒烟这件事变成每天都能看到的进步：

记录每天是否打卡
统计已经坚持了多少天
按打卡次数计算省下多少钱
展示吸烟危害与身体恢复时间线
首页按日期轮换展示一句鼓励文案
新版首页用更强的视觉层次，把“进步、金额、连续天数”第一眼就亮出来

2. 核心特点
   
数据保存在本机：开始日期、每日烟钱、每日支数、打卡记录都会保存在当前设备本地。
按日历方式打卡：可以直接在日历里查看和管理每天的打卡状态。
自动计算省钱金额：按“打卡次数 × 每天烟钱”计算累计节省金额。
可视化查看进度：首页、打卡页、肺部对比页、恢复时间线页一目了然。
鼓励式首页文案：首页会展示一句每日文案，帮助保持节奏。

正在开始戒烟的人

希望看到“坚持带来了什么变化”的人
想通过省钱统计增强成就感的人
